## 📋 项目概览

### [Hermes Agent 记忆系统](hermes-agent/)

**Hermes Agent** 是由 Nous Research 开发的自主 AI 代理系统。

- **记忆系统** - 多层次记忆架构（Session/Memory/Skills）
- **技能系统** - 程序性知识管理与自动发现
- **上下文系统** - 项目感知与渐进式披露

📖 [详细文档](hermes-agent/README.md)

### [OpenAI Privacy Filter](openai-privacy-filter/)

**OpenAI Privacy Filter** 是一个 1.5B 参数的 MoE 模型，用于检测和脱敏文本中的 PII。

- **参数量**: 1.5B 总参数，50M 激活参数
- **功能**: PII 检测、文本脱敏、批量处理
- **测试**: 个人敏感信息 + 复杂.env 文件测试
- **检测率**: ~75%

📖 [详细文档](openai-privacy-filter/README.md)
